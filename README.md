# iPhotoAlbum <img src="ICON64.png">
<h3>
  This app make album photo lists for prepare backup album.
  <hr>
  <p>Prepare Backup Structure</p>
  By Monthly：
  1. Choose Album
  2. Choose Year
  3. Choose Monthly
  4. Make List
  5. Upload List (AirTrop to Mac)

  By Yearly：
  1. Choose Album
  2. Choose Year
  3. Make List (Swipe Left)
  4. Upload List (AirTrop to Mac)
